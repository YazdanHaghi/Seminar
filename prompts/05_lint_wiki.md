# Prompt: Lint the Wiki

Check the LLM Wiki for quality problems.

Find:

- duplicate pages
- broken links
- outdated information
- contradictions
- missing source references
- pages that should be merged
- topics that need overview pages
- commands without explanation
- benchmark results without hardware/compiler context

Return:

1. Problems found
2. Suggested fixes
3. Pages to create
4. Pages to merge
5. Highest priority cleanup tasks
